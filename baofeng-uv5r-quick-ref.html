Manual Programming Guide VFO Mode Required
Method A: Simplex (Direct/Scanner)

Use for Walkie-Talkie channels, Weather, or Railroad.

  1. Press VFO/MR to see Frequency Mode.
  2. Type frequency (e.g., 462.562).
  3. Clear Offsets: Ensure Menu 25 and 26 are OFF/000.
  4. Store:
    Press MENU 2 7 MENU.
  5. Select Channel (e.g., 001) using .
  6. Press MENU to Confirm. Press EXIT.
Method B: Repeater (Relay)

Use for Ham Radio Repeaters (Requires PL Tone & Offset).

  1. Freq: Type Output Freq (e.g., 146.520).
  2. Tone (Tx): MENU 1 3 MENU → Select Tone → MENU.
  3. Direction: MENU 2 5 MENU → Select + or - → MENU.
  4. Offset: MENU 2 6 MENU → Type offset (e.g., 000600) → MENU.
  5. Save: MENU 2 7 MENU → Select Channel → MENU.
Note: When you press PTT, the screen frequency will change. This is normal (Offset is working).

Troubleshooting (Manual Page 18)
Radio won't start?
Battery is dead or not installed correctly (Check contacts).
Light is green, but no sound?
1. Check Volume knob.
2. Check Menu 11 (R-CTCS) is OFF.
3. Check Menu 12 (R-DCS) is OFF.
Others can't hear you?
1. Check Menu 13 (T-CTCS) matches repeater.
2. Check offset direction/amount.
3. Are you too far away? (Line of Sight).
Can't type frequency?
Example: "It jumps from 146.520 to 146.525!"
Fix: Change Menu 1 (STEP) to 2.5K or 5.0K.
