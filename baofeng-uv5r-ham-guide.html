Hardware & Setup
Side Keys
  • CALL (Top): FM Radio. Hold for ALARM.
  • MONI (Bot): Flashlight. Hold to Open Squelch (Listen).
  • PTT: Push To Talk.
Front Keys
  • VFO/MR Switch Freq/Channel Mode.
  • A/B Switch Active Line (Arrow).
  • BAND Switch VHF/UHF.
  • *SCAN Hold to Scan. Tap for Reverse.
The "Golden 6" Settings

Configure these first (Menu → # → Menu → Value → Menu).

0 SQL Squelch 5 (Filters static)
1 STEP Tuning Step 5.0K (Standard)
2 TXP Power HIGH (8W) / LOW (1W)
5 W/N Bandwidth WIDE (Most Ham/GMRS)
7 TDR Dual Watch ON (Listen to A & B)
13 T-CTCS Tx Tone OFF (Set per channel)
Programming Guide
Prerequisite: Press VFO/MR to enter "Frequency Mode" (No channel numbers on right).
  • Standard Repeater
  • Simplex/Scanner
Programming a Repeater (e.g., SkyHub)
1. Frequency: Type 449.450 (Example).
2. Tone (Key): MENU 13 (T-CTCS) → Select 103.5 → MENU .
NOTE: Keep Menu 11 (R-CTCS) OFF unless needed.
3. Offset Dir: MENU 25 (SFT-D) → Select - (MINUS).
4. Offset Amt: MENU 26 (OFFSET) → Type 005.000 .
5. Save: MENU 27 (MEMCH) → Select Channel → MENU .
Programming Simplex (Direct) or Scanning

For NOAA, Railroad, or Walkie-Talkie channels.

1. Frequency: Type frequency (e.g., 162.550 ).
2. Clean Up: Ensure Menu 25 (SFT-D) is OFF .
3. Save: MENU 27 (MEMCH) → Select Channel → MENU .
The "Split Tone" Trap (WA0MJX): Some repeaters transmit a different tone than they receive. If you can't hear audio, ensure Menu 11 (R-CTCS) is OFF . Only set Menu 13.
Ham Radio Procedures

Example: Sunday 8:30 AM on 145.310 (ColCon).

  1. Listen First: Wait for "Net Control" to ask for "Check-ins."
  2. Wait for your turn: They often ask by alphabet ("Alpha through Foxtrot").
  3. Press PTT: "This is [Your Callsign], [Your Name], No Traffic."
  4. Release PTT: Listen for acknowledgement.

Emergency / Disaster Plan
1. The Golden Rule

If you have a cell signal, use it. Radio is a backup. If 911 is down, use Linked Repeaters (SkyHub/ColCon) first.

2. The Script (Don't Ramble)
Life Threat: "MAYDAY, MAYDAY, MAYDAY. This is [Callsign]. I have a life threatening emergency at [Location]."
Stranded/Injured: "BREAK, BREAK, BREAK. This is [Callsign] with emergency traffic."
3. Disaster News Sources (Listen Only)
  • KOA News: 94.1 MHz (FM Radio mode).
  • Media IFB: 450.1625 MHz (Hear reporters off-air).
  • Railroads: 161.100 MHz (Track conditions).

FCC Regulation: In an immediate threat to life/property where no other means exist , you may use ANY frequency to call for help.
Suggestion: If Ham fail, try GMRS Ch 1 (462.5625) or MURS Ch 1 (151.820).
Walkie-Talkie Mode

The UV-5R can communicate with blister-pack walkie talkies (Motorola/Midland) found at stores.
Note: Technically requires GMRS license and Part 95 hardware, but widely used in emergencies.

*Set STEP to 2.5K or 6.25K for these frequencies.
