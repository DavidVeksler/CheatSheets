EMERGENCY RADIO GUIDE

IF PHONES ARE DOWN, USE THIS TO CALL FOR HELP.

1
Turn On / Volume
Twist top knob clockwise. Turn volume up halfway.
2
Select Channel
Use ▲ ▼ arrows to find Channel 001.
HOW TO CALL FOR HELP
PTT BUTTON
(LEFT SIDE)
  1. HOLD DOWN the side button. Wait 1 second.
  2. SPEAK SLOWLY into the front hole.
  3. RELEASE button to listen.
READ THIS EXACTLY:
"MAYDAY, MAYDAY, MAYDAY."
"MY NAME IS    [YOUR NAME]   ."
"I AM LOCATED AT    [YOUR ADDRESS]   ."
"WE HAVE AN EMERGENCY:    [FIRE / INJURY]   ."
"PLEASE SEND HELP."
(Now release button and listen)

TROUBLESHOOTING: If keys don't work, hold # for 2 seconds to UNLOCK. If screen is dark, tap any key.
OPERATOR SETUP REQUIRED:
  • Ch 001: Program local repeater (SkyHub Link 449.450 T:103.5 -5.0).
  • Ch 002: Program NOAA Weather (162.550).
  • Lock: Put radio in Channel Mode, then hold # to lock keypad.
  • Print: Use Browser Print (Ctrl+P). Set Margins to "Minimum" or "None".
